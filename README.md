# 软考项目管理 - 智能做题系统

## 📚 项目简介

这是一个专为软考(软件水平考试)项目管理科目设计的现代化在线做题应用。采用纯前端技术栈,无需服务器,所有数据存储在浏览器本地,保护您的学习隐私。

## ✨ 核心功能

### 🎯 学习功能

- **章节练习**: 按照教材章节系统化学习,循序渐进
- **随机练习**: 随机抽取题目,全面检验知识掌握
- **错题本**: 自动收集错题,针对性复习薄弱环节
- **收藏夹**: 收藏重点题目,方便反复练习
- **搜索功能**: 快速查找特定知识点的题目

### 📊 统计分析

- **学习进度**: 实时显示各章节完成情况
- **正确率统计**: 精确计算答题正确率
- **学习天数**: 记录累计学习天数,激励持续学习
- **章节分析**: 详细展示每个章节的掌握程度

### 🎨 用户体验

- **深色/浅色主题**: 支持主题切换,保护视力
- **响应式设计**: 完美适配桌面、平板和手机
- **流畅动画**: 现代化的交互动画,提升使用体验
- **本地存储**: 所有数据保存在本地,无需联网

## 🚀 快速开始

### 方法一: 直接打开(推荐)

1. 双击 `index.html` 文件
2. 使用浏览器打开即可开始使用

### 方法二: 本地服务器

如果需要更好的体验,可以使用本地服务器:

```bash
# 使用 Python 3
python -m http.server 8000

# 使用 Node.js (需要先安装 http-server)
npx http-server -p 8000

# 然后在浏览器访问
http://localhost:8000
```

## 📖 使用指南

### 首次使用

1. 打开应用后,首页会显示学习统计概览
2. 点击"章节练习"选择要学习的章节
3. 开始答题,系统会自动记录您的学习进度

### 答题流程

1. 阅读题目内容
2. 选择您认为正确的答案
3. 点击"提交答案"查看结果
4. 阅读答案解析,理解知识点
5. 使用"上一题/下一题"继续练习

### 功能说明

#### 章节练习

- 选择特定章节进行系统学习
- 显示章节完成进度和正确率
- 适合第一遍学习和系统复习

#### 随机练习

- 从题库中随机抽取题目
- 适合考前冲刺和全面检验
- 每次练习10道题

#### 错题本

- 自动收集答错的题目
- 支持重新练习错题
- 答对后自动从错题本移除

#### 收藏夹

- 点击题目页面的星标图标收藏
- 收藏重点题目和易错题
- 方便考前快速复习

#### 学习统计

- 查看详细的学习数据分析
- 了解各章节掌握情况
- 制定针对性学习计划

#### 图片查看功能

题目中的图片支持高级查看功能:

- **点击图片**: 打开全屏模态框查看
- **滚轮缩放**: 向上滚动放大(最大5倍),向下滚动缩小(最小0.5倍)
- **拖拽移动**: 放大后可以拖拽图片查看不同区域
- **双击重置**: 双击图片恢复到原始大小和位置
- **关闭方式**:
  - 点击右上角 × 按钮
  - 点击背景区域
  - 按 ESC 键
- **智能适配**: 图片自动适应页面宽度,不会撑坏布局

#### 多空题功能

系统支持多空题（一题多问）类型，提供更灵活的答题体验：

- **题型识别**: 自动识别包含多个空格的题目
- **独立作答**: 每个空格都有独立的选项下拉框
- **分步提交**: 可以逐个空格填写，也可以一次性提交所有答案
- **精确判分**: 系统会分别判断每个空格的正确性
- **详细反馈**: 提交后会显示每个空格的对错情况
- **完整解析**: 答案解析会针对所有空格进行说明
- **适用场景**: 特别适合综合性题目和案例分析题

**示例格式**：

```
题目：某项目的关键路径为（ 1 ），总工期为（ 2 ）天。
选项1：A-B-C / A-D-E / B-C-D / C-D-E
选项2：10 / 15 / 20 / 25
```

## 🎯 题目说明

### 当前题库

应用中包含示例题目,涵盖以下章节:

- 第1-3章: 信息化基础
- 第四章: 英语词汇
- 第五章: 选择题技巧
- 第13-25章: 项目管理核心知识

### 题型分类

系统支持多种题型，满足不同的考试需求：

#### 单选题 (Single Choice)

- 标准的单项选择题
- 4个选项中选择1个正确答案
- 最常见的题型

#### 多空题 (Multiple Slots)

- 一题多问，包含多个空格
- 每个空格独立选择答案
- 适合综合性题目和案例分析
- 系统会分别判断每个空格的正确性

**题型标识**：

- `type: 'single'` - 单选题
- `type: 'multiple'` 或包含 `slots` 数组 - 多空题

### 添加更多题目

要添加更多题目,请编辑 `data.js` 文件:

```javascript
// 单选题示例
{
    id: 'q011',
    chapterId: 'ch13',
    type: 'single',
    difficulty: 'medium',
    content: '题目内容',
    options: ['选项A', '选项B', '选项C', '选项D'],
    correctAnswer: 0,  // 正确答案索引(从0开始)
    explanation: '答案解析',
    tags: ['标签1', '标签2']
}

// 多空题示例
{
    id: 'q012',
    chapterId: 'ch13',
    type: 'multiple',
    difficulty: 'medium',
    content: '某项目的关键路径为（ 1 ），总工期为（ 2 ）天。',
    slots: [
        {
            label: '空格 1',
            options: ['A-B-C', 'A-D-E', 'B-C-D', 'C-D-E'],
            answer: 'B'  // 对应第2个选项
        },
        {
            label: '空格 2',
            options: ['10', '15', '20', '25'],
            answer: 'C'  // 对应第3个选项
        }
    ],
    explanation: '答案解析...',
    tags: ['进度管理', '关键路径']
}
```

## 💾 数据管理

### 数据存储

所有学习数据存储在浏览器的 localStorage 中,包括:

- 答题记录
- 学习进度
- 错题列表
- 收藏题目
- 学习统计

### 数据导出

打开浏览器控制台(F12),执行以下代码导出数据:

```javascript
// 导出所有数据
const data = {
    progress: localStorage.getItem('exam_user_progress'),
    answers: localStorage.getItem('exam_user_answers'),
    favorites: localStorage.getItem('exam_favorites'),
    mistakes: localStorage.getItem('exam_mistakes'),
    stats: localStorage.getItem('exam_study_stats')
};
console.log(JSON.stringify(data, null, 2));
```

### 数据清除

如需重置所有学习数据,在浏览器控制台执行:

```javascript
localStorage.clear();
location.reload();
```
